## 23.合并k个升序链表
- **标签：** 链表/归并排序
- **原题链接：** [点击跳转](https://leetcode.cn/problems/merge-k-sorted-lists/description/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
对归并中分治的写法还是很不熟悉，t148因为其是一整条链表，自上而下时所以采用找中点的方法。而本题这些链表放在数组里，其实完全可以用数组的逻辑，代码见下。
### 💻 C++ 自上而下，递归分治代码实现
```cpp
 class Solution {
private:
    ListNode* merge(ListNode* l1,ListNode* l2){
        ListNode dummy(0);
        ListNode* cur = &dummy;
        while(l1&&l2){
            if(l1->val<=l2->val){
                cur->next = l1;
                l1 = l1->next;
            }
            else{
                cur->next = l2;
                l2 = l2->next;
            }
            cur = cur->next;
        }
        cur->next = l1?l1:l2;
        return dummy.next;
    }
    ListNode* slove(vector<ListNode*>& lists,int left,int right){
        if(left == right) return lists[left];
        if(left>right) return nullptr;
        int mid = left+(right-left)/2;
        ListNode* l1 = slove(lists,left,mid);
        ListNode* l2 = slove(lists,mid+1,right);
        return merge(l1,l2);
    }
public:
    ListNode* mergeKLists(vector<ListNode*>& lists) {
        if(lists.size()==0) return nullptr;
        return slove(lists,0,lists.size()-1);
    }
};
```
### 🔥值得注意
其实一开始想的就是直接在原vector上操作，但每次合并后的链表放在哪呢，我的想法时移到前面来，结果奇数时会有额外操作，而且很不好控制，便放弃了。其实还真可以原地操作，非常巧妙。

### 💻 C++ 自下而上，迭代归并代码实现
```cpp
class Solution {
private:
    ListNode* merge(ListNode* l1,ListNode* l2){
        ListNode dummy(0);
        ListNode* cur = &dummy;
        while(l1&&l2){
            if(l1->val<=l2->val){
                cur->next = l1;
                l1 = l1->next;
            }
            else{
                cur->next = l2;
                l2 = l2->next;
            }
            cur = cur->next;
        }
        cur->next = l1?l1:l2;
        return dummy.next;
    }
public:
    ListNode* mergeKLists(vector<ListNode*>& lists) {
        int k = lists.size();
        if(k==0) return nullptr;
        //两层for循环完美处理了逻辑
        for(int interval = 1;interval<k;interval<<=1){         
            for(int i = 0;i+interval<k;i+=(interval<<1)){
                lists[i] = merge(lists[i],lists[i+interval]);
            }
        }
        return lists[0];
    }
};
```
### 🔥值得注意
还可以用堆，很类似外部排序的感觉，一个个往里面放。
```cpp
class Solution {
public:
    // 定义一个比较器：因为我们要小顶堆，所以是 val1 > val2
    struct compare {
        bool operator()(ListNode* l, ListNode* r) {
            return l->val > r->val; 
        }
    };

    ListNode* mergeKLists(vector<ListNode*>& lists) {
        // 1. 创建小顶堆
        priority_queue<ListNode*, vector<ListNode*>, compare> pq;

        // 2. 把所有链表的头节点扔进去（注意判空）
        for (auto head : lists) {
            if (head) pq.push(head);
        }

        ListNode dummy(0);
        ListNode* cur = &dummy;

        // 3. 循环弹出并压入
        while (!pq.empty()) {
            ListNode* node = pq.top();
            pq.pop();
            
            cur->next = node;  // 接入结果
            cur = cur->next;

            if (node->next) {  // 替补进堆
                pq.push(node->next);
            }
        }
        return dummy.next;
    }
};
```
