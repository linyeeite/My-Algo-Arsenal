## 101.对称二叉树
- **标签：** 二叉树
- **原题链接：** [点击跳转](https://leetcode.cn/problems/symmetric-tree/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
有模糊的比较外侧和内侧的思路，但很薄弱，还是没想出来，卡在了如何传递参数上，因为原函数就给一个参数，单个参数根本无法分内侧外侧，自然我们得自己搞个新函数。然后处理好返回逻辑即可。
### 💻 C++ 递归极简代码实现
```cpp
 class Solution {
public:
    bool check(TreeNode *p, TreeNode *q) {
        if (!p && !q) return true;
        if (!p || !q) return false;
        return p->val == q->val && check(p->left, q->right) && check(p->right, q->left);
    }

    bool isSymmetric(TreeNode* root) {
        return check(root->left, root->right);
    }
};
```
### 🔥值得注意
迭代法，肯定是用队列，after all是一行一行比的，但我一开始想的是不将不合法的点和空结点放进入队列里，所以在放入前就要判断很多东西，**事实上完全可以将nullptr放进队列**，出来才统一判断，代码简洁很多。见下即可。

### 💻 C++ 迭代复杂代码实现
```cpp
class Solution {
public:
    
    bool isSymmetric(TreeNode* root) {
       queue<TreeNode*> que;
       if(!root->left&&!root->right) return true;
       if(!root->left||!root->right) return false;
       que.push(root->left);
       que.push(root->right);
       while(!que.empty()){
            TreeNode* node1 = que.front();que.pop();
            TreeNode* node2 = que.front();que.pop();
            if(node1->val!=node2->val) return false;
            if(node1->left&&node2->right) {
                que.push(node1->left);
                que.push(node2->right);
            }
            if(node1->left&&!node2->right) return false;
            if(!node1->left&&node2->right) return false;
            if(node1->right&&node2->left) {
                que.push(node1->right);
                que.push(node2->left);
            }
            if(node1->right&&!node2->left) return false;
            if(!node1->right&&node2->left) return false;
       }
       return true;
    }
};
```

### 💻 C++ 迭代简洁代码实现
```cpp
class Solution {
public:
    bool isSymmetric(TreeNode* root) {
       queue<TreeNode*> que;
       if(root == nullptr) return true;
       else { 
        que.push(root->left);
        que.push(root->right);
       }
        while(!que.empty()){
            TreeNode* leftnode = que.front();que.pop();
            TreeNode* rightnode = que.front(); que.pop();
            if(!leftnode&&!rightnode) continue;
            if(!leftnode||!rightnode||leftnode->val!=rightnode->val) return false;

            que.push(leftnode->left);
            que.push(rightnode->right);
            que.push(leftnode->right);
            que.push(rightnode->left);
        }
        return true;
    }
};
```
