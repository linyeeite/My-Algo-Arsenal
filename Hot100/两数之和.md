## 1. 两数之和
- **标签：** 哈希表 / 数组
- **原题链接：** [点击跳转](https://leetcode.cn/problems/two-sum/)

### 💡 思路历程
一开始还在想回溯，那不就是暴力吗，但怎么也没把哈希表想到，其实使用哈希表记录已经遍历过的数字及其索引，用空间换时间就ok了，因为无序的map查询的时间和增删的效率都是O（1）。

### 💻 C++ 代码实现
```cpp
class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        unordered_map<int,int> umap;
        for(int i = 0;i<nums.size();i++){
            if(umap.find(target-nums[i])!=umap.end()){
                return{umap[target-nums[i]],i};
            }
            umap.insert(pair<int,int>(nums[i],i));
        }
        return {};
    }
};
```
### 值得注意
map的插入语法，用pair对，或者umap.insert({nums[i],i});  或者umap.emplace(nums[i],i);emplace是更好的。

为什么 emplace 可能更好？：

原地构造：insert 通常需要先创建一个 pair 临时对象，再把它“拷贝”或“移动”到哈希表里。

效率：emplace 直接在哈希表的内存空间里构造这个元素，省去了临时对象的开销。虽然对于 int 这种基础类型差别很小，但如果处理的是复杂的 AI 模型参数对象，性能差距会非常明显。
