## 15. 三数之和
- **标签：** 双指针 / 数组
- **原题链接：** [点击跳转](https://leetcode.cn/problems/3sum/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
时隔两月，可谓是忘得一干二净，再重述下思路吧，首先哈希法去重逻辑太为复杂，不考虑，用双指针，由于题目求得是数字，所以我们直接排序即可，等于0的逻辑并不困难，我们主要看去重逻辑：第一个数的去重，要注意到是允许单个满足的数组里能有重复的元素，只是不允许有重复的数组
，所以if判断就有说法，见代码。然后是left和right的去重，为防止还没取到就被去重，去重的逻辑应该放在将一个满足题意的一维数组放入result里时在进行去重，还要注意的是，只有找到符合题意的数组时，left和right才会再收缩以此，收缩逻辑的位置别写错了。

### 💻 C++ 代码实现
```cpp
class Solution {
public:
    vector<vector<int>> threeSum(vector<int>& nums) {
        vector<vector<int>> result;
        sort(nums.begin(),nums.end());
        
        for(int i = 0;i<nums.size();i++){
            if(nums[i]>0) return result;
            if(i>0&&nums[i]==nums[i-1]) continue;
            int left = i+1;
            int right = nums.size()-1;
            while(left<right){
                if(nums[left]+nums[right]+nums[i]>0){
                    right--;
                }
                else if(nums[left]+nums[right]+nums[i]<0){
                    left++;
                }
                else{
                    result.push_back({nums[i],nums[left],nums[right]});
                    while(left<right&&nums[left]==nums[left+1]) left++;
                    while(left<right&&nums[right-1]==nums[right]) right--;
                    left++;
                    right--;
                }
                
            }
        }
        return result;
    }
};
```
### 值得注意
四数之和同理，多层for循环，如果target不是0，自然去重逻辑和指针移动逻辑稍许变化，不过也简单。注意第二个指针的起始位置为i+1。
