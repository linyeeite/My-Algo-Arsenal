## 239.滑动窗口的最大值
- **标签：** 单调队列/数组
- **原题链接：** [点击跳转](https://leetcode.cn/problems/sliding-window-maximum/solutions/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
还是只能想到暴力解法，然而像这种要找滑动窗口最大值，用单调队列很方便，自定义pop与push逻辑达成题目要求。
### 💻 C++ 代码实现
```cpp
class Solution {
private:
    class MyQueue{
    public:
       deque<int> que;
       void pop(int val){
            if(!que.empty()&&que.front()==val){
                que.pop_front();
            }
       }
       void push(int val){
            while(!que.empty()&&val>que.back()){
                que.pop_back();
            }
            que.push_back(val);
       }
       int front(){
            return que.front();
       }
    };

public:
    vector<int> maxSlidingWindow(vector<int>& nums, int k) {
        MyQueue que;
        vector<int> result;
        for(int i = 0;i<k;i++){
            que.push(nums[i]);
        }
        result.push_back(que.front());
        for(int i = k;i<nums.size();i++){
            que.pop(nums[i-k]);
            que.push(nums[i]);
            result.push_back(que.front());
        }
        return result;

    }
};
```
### 🔥值得注意
在这里自述一下本体逻辑。使用单调队列，队头为对应窗口的最大值，要适应滑动窗口的移动，我们要自定义一个队列，即用deque这个底层容器，队列其实只要放可以成为最大元素的值就好了。

push时，如果本元素大于前面的元素，自然弹出前面的，**直到碰到不比自己小的**，然后再把自己push_back,即放到队尾。

pop时，因为我们在push时是有选择的push的，有些已经被弹出了，所以一个很自然的逻辑就是，只有队头元素是自己时，才弹出，说明窗口往前移动，本元素已经不在窗口里了，而本元素又是该窗口未移动前的最大元素，故弹出。这些逻辑用if判断就好了，使得后续逻辑好写。

再来个取队头元素的函数即可。

然后初始化先将前k个元素放入队列，将队头元素放入结果里，然后才从第k+1个开始，pop第前k个元素，push当前元素，再取队头元素加入结果集。遍历完即可。
