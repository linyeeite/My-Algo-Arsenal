## 25.K个一组翻转链表
- **标签：** 链表
- **原题链接：** [点击跳转](https://leetcode.cn/problems/reverse-nodes-in-k-group/description/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
手撕成功，用的是迭代法，递归确实没能想明白，但其实跟两两交换那题非常相似，也有点像先序遍历和中序遍历那样，先处理本段逻辑，再处理下面逻辑，还是自上向下的，链表的话在于原本head结点会成为本段最后一个结点，然后要接后面处理的链表，很适合递归逻辑。

### 💻 C++ 迭代法代码实现
```cpp
class Solution {
private: 
    bool is_vaild(ListNode* head,int k){
        ListNode* cur = head->next;
        while(k--){
            if(cur==nullptr) return false;
            cur = cur->next;
        }
        return true;
    }
    ListNode* reverse(ListNode* head,int k){
        ListNode *cur = head->next,*Next;
        ListNode* last = cur;
        head->next = nullptr;
        while(k--){
            Next = cur->next;
            cur->next = head->next;
            head->next = cur;
            cur = Next;
        }
        last->next = Next;
        return last;
    }
public:
    ListNode* reverseKGroup(ListNode* head, int k) {
        ListNode* dummy = new ListNode();
        dummy->next = head;
        ListNode* cur = dummy;
        while(is_vaild(cur,k)){
            cur = reverse(cur,k);
        }
        head = dummy->next;
        delete(dummy);
        return head;
    }
};
```
### 💻 C++ 递归法代码实现
```cpp
class Solution {
public:
    ListNode* reverseKGroup(ListNode* head, int k) {
       ListNode* cur = head;
       for(int i = 0;i<k;i++){
            if(cur == nullptr) return head;
            cur = cur->next;
       }
       ListNode* pre = nullptr;
       ListNode* node = head;
       for(int i = 0;i<k;i++){
            ListNode* next = node->next;
            node->next = pre;
            pre = node;
            node = next;
       }
       head->next = reverseKGroup(node,k);
       return pre;
    }
};
```
