## 295.数据流的中位数
- **标签：** 堆
- **原题链接：** [点击跳转](https://leetcode.cn/problems/find-median-from-data-stream/description/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
想法是用小顶堆，然后加元素调整堆的时间是logn，如果要求中位数，我们知道堆的大小，通过vector容器也能很好的找到中位数，但是，我的疑惑点在于如果直接用优先队列，他底层可以是vector，但怎么写呢，能做到随机访问吗？

答案如下：堆的底层数组并不是有序的。即便底层是 vector，C++ 的 std::priority_queue 在封装时并没有暴露随机访问迭代器.

所以本题思路是用两个堆，一个大顶堆用来记录一半相对小的元素，一个小顶堆记录一般相对大的元素，然后保持元素个数不超过一，至于是谁会多一个都可以。因为优先队列我们访问top轻而易举，此时中位数也显而易见了。
### 💻 C++ 代码实现
```cpp
class MedianFinder {
public:
    priority_queue<int,vector<int>,less<int>> queMax;
    priority_queue<int,vector<int>,greater<int>> queMin;
    
    MedianFinder() {

    }
    
    void addNum(int num) {
        if(queMax.empty()||num<queMax.top()){
            queMax.push(num);
            if(queMin.size()+1<queMax.size()){
                queMin.push(queMax.top());
                queMax.pop();
            }
        }
        else{
            queMin.push(num);
            if(queMin.size()>queMax.size()){
                queMax.push(queMin.top());
                queMin.pop();
            }
        }
    }
    
    double findMedian() {
        if(queMax.size()>queMin.size()) return queMax.top();
        return (queMax.top()+queMin.top())/2.0;
    }
};


```
