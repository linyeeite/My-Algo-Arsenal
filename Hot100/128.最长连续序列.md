## 128.最长连续序列 
- **标签：** 哈希表 / 数组
- **原题链接：** [点击跳转](https://leetcode.cn/problems/longest-consecutive-sequence/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
想到用set，也想到了如果对每个数字都往后找的话，时间上肯定不是O(n)，但就差那一招，没想到可以如此：如果前一个数存在，自然就跳过就好了！结果还是有小差错，见下文

### 💻 C++ 首版代码实现
```cpp
class Solution {
public:
    int longestConsecutive(vector<int>& nums) {
        unordered_set<int> uset;
        int result = 0;
        for(int s : nums){
            uset.emplace(s);
        }
        for(int i = 0;i<nums.size();i++){
            int cur = nums[i];
            int count = 0;
            if(!uset.contains(cur-1)){
                while(uset.contains(cur)){
                    count++;
                    cur++;
                }
                result = max(result,count);
            } 
        }
        return result;
    }
};
```
### 💡 错因
遍历了nums中的每个数，如果nums有很多重复的数字，会导致多次哈希查找，从而超时，其次，如果nums中有数刚好是INT_MAX，int会越界，所以非常robust的代码见下文。

### 💻 C++ 首版代码实现
```cpp
class Solution {
public:
    int longestConsecutive(vector<int>& nums) {
        // 1. 利用哈希表去重并提供 O(1) 查找
        unordered_set<long long> uset(nums.begin(), nums.end());
        int result = 0;

        // 2. 直接遍历去重后的集合，效率更高
        for (long long num : uset) {
            // 3. 只有当 num 是序列的起点时才开始计数（数学上的下确界）
            if (!uset.count(num - 1)) {
                long long cur = num;
                int count = 0;

                while (uset.count(cur)) {
                    count++;
                    cur++;
                }
                result = max(result, count);
            }
        }
        return result;
    }
};
```

### 值得注意
uset的操作函数，find(val),count(val),insert(val),emplace(val),contains(val).
