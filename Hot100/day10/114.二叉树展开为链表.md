## 114.二叉树展开为链表
- **标签：** 二叉树
- **原题链接：** [点击跳转](https://leetcode.cn/problems/flatten-binary-tree-to-linked-list/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
没能想到如何原地操作，用了栈进行前序遍历，再用队列记录前序遍历的顺序，然后依次改变左右节点。实际上通过找左结点的最右边结点，即可完成原地操作。在此重述逻辑：

1.若本结点left为nullptr，那遍历right即可。

2.反之，找到左孩子的最右下结点，（即原结点的右节点的前驱结点）括号内的说法是错误的！并非一定是右节点的前驱节点，因为该右节点可能有左孩子，但其实无关系，后续遍历到这时会将左孩子放到后面去。

3.然后将最右下结点的右指针指向原结点的右孩子，再将原结点的左孩子接到右指针，然后将左指针置空。

4.往右边遍历。
### 💻 C++ 递归极简代码实现
```cpp
 class Solution {
public:
    void flatten(TreeNode* root) {
       TreeNode* cur = root;
       while(cur){
            if(cur->left){
                TreeNode* pre = cur->left;
                while(pre->right){
                    pre = pre->right;
                }
                pre->right = cur->right;
                cur->right = cur->left;
                cur->left = nullptr;
            }
            cur = cur->right;  
       }
       return;
    }
};
```
