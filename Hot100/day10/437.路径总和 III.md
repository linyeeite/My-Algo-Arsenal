## 437.路径总和 III
- **标签：** 二叉树/回溯
- **原题链接：** [点击跳转](https://leetcode.cn/problems/path-sum-iii/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
一开始只能想到双重递归，我用了一个很巧妙的flag来判断是否能将当前sum值不经减本结点值而往下传，ac了，而且速度其实比标准的二重递归快的。但本题最好的做法还是前缀和+哈希表，有点像二数之和，但其实比二数之和难很多。在这里叙述下思路：

1.首先前缀和指的是从**根节点开始向下**，到某一结点的前缀和，然后记录在prefix里。

2.如果某一路径的和为targetsum，说明prefix里存在对应前缀和（不妨设为prevSum），使得targetsum-cursum 等于 prevSum。 其中cursum也是从根节点到本结点这一路径的和。

推导如下：curSum - prevSum = targetSum  $\Rightarrow$  prevSum = currSum - targetSum 

我们利用这个前缀和就非常好地完成了题目的要求，即路径可以不一定从根结点出发，也不一定在叶子节点结束。如果不用双重递归其实很难实现这种要求，但我们利用前缀和（利用其连续的性质，即不会出现中间有断结点的性质），在一次递归中记录好从根节点开始的前缀和，通过cursum-presum  = targetsum很好地完成了这个要求。

手动跑一下就懂了。

3.最后一定要注意回溯，即对前缀和的回溯，因为这个前缀和只能贡献给它的子孙节点，不能贡献给它的“兄弟”或“叔叔”节点，所以return前得将prefix里对应的前缀和数量减一。

### 💻 C++ 双重递归flag代码实现
```cpp
 class Solution {
public:
    int count = 0;
    void traversal(TreeNode* root,long long sum,bool flag){
        if(sum == root->val) count++;
        if(!root->left&&!root->right) return;
        if(root->left){
            traversal(root->left,sum-root->val,false);
            if(flag)traversal(root->left,sum,true);
        }
        if(root->right){
            traversal(root->right,sum-root->val,false);
            if(flag)traversal(root->right,sum,true);
        }
    }
    int pathSum(TreeNode* root, int targetSum) {
        if(!root) return 0;
        traversal(root,targetSum,true);
        return count;
    }
};
```

### 💻 C++ 前缀和代码实现
```cpp
 class Solution {
public:
    unordered_map<long long,int> prefixMap;

    int dfs(TreeNode* root,long long cursum,int targetSum){
        if(!root) return 0;
        int res = 0;
        cursum +=root->val;
        if(prefixMap.count(cursum-targetSum)){
            res += prefixMap[cursum-targetSum];
        }
        prefixMap[cursum]++;
        res += dfs(root->left,cursum,targetSum);
        res += dfs(root->right,cursum,targetSum);
        prefixMap[cursum]--;
        return res;
    }
    int pathSum(TreeNode* root, int targetSum) {
        prefixMap.clear();
        prefixMap[0] = 1;
        return dfs(root,0,targetSum);
    }
};
```
