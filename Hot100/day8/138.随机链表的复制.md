## 138.随机链表的复制
- **标签：** 链表
- **原题链接：** [点击跳转](https://leetcode.cn/problems/copy-list-with-random-pointer/description/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
一开始想迭代去做，发现random指针指向的结点还没创建出来怎么办呢，于是想先遍历一边链表，把next都弄出来，在这个过程中，用map<node*,node*>记录对应指针，然后接上就行，但其实我一开始没有想到用这样的map，我一直在想记录int的map，有点男泵。

其次，递归其实很优雅，但也得用到map，防止重复创建新节点。对于O（1）空间复杂度的迭代法，要明确一点，为了记录好random指针的指向，遍历的过程中怎么都得记录random信息，不准用map，那我们就在原链表上记录，怎么记录呢？

我们将原链表每个结点的duplication就放在各自结点后，并与后面的结点相连接，这里代码要注意指针的走向关系，否则容易出错，构建好后，random指针就很好指向了，还是很优雅的，最后再断开，指针关系同样得处理好。
### 💻 C++ 迭代代码实现
```cpp
class Solution {
public: 

    Node* copyRandomList(Node* head) {
        if(head == NULL) return NULL;
        Node* cur = head;
        while(cur){
            Node* newnode = new Node(cur->val);
            newnode->next  = cur->next;
            cur->next = newnode;
            cur = newnode->next;
        }
        cur = head;
        while(cur){
            if(cur->random){
                cur->next->random = cur->random->next;
            }
            cur = cur->next->next;
        }
        cur = head;
        Node* dummy = new Node(0);
        Node* curr = dummy;
        while(cur){
            curr->next = cur->next;
            curr = curr->next;

            cur->next = curr->next;
            cur = cur->next;
        }
        curr = dummy->next;
        delete(dummy);
        return curr;
    }
};
```
