## 148.排序链表
- **标签：** 链表
- **原题链接：** [点击跳转](https://leetcode.cn/problems/sort-list/description/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
只能想到数组，根本想不到归并，想到归并，也只能想到自上而下，同样有logn的空间复杂度。然后对于链表的递归，其实也很不熟，在此叙述以下逻辑：

首先，归并排序是递归的，从两个元素排起，然后逐渐扩大。需要三个函数，第一个为主函数，第二个函数为找中点，然后调用第三个merge函数，是递归体，merge函数即简单归并。

细节在于第二个函数，模板可以理解为**左闭右开区间**，不断往下分，直到分到只有一个元素为止，排序返回指针继续排序。最好是自己模拟一边。

还有找中点时，与我之前习惯的写法不一致，本题找的中点并非实际意义上的中点，这是因为我们的区间是左必右开的缘故，但它并不像我们之前找中点为了断开，这个点的位置实际上为前半段提供tail作为边界控制，为后半段提供左区间用于对比排序。

所以实际上并不能以常规意义看待这个中点。
### 💻 C++ 自上而下归并代码实现
```cpp
class Solution {
public:
    ListNode* sortList(ListNode* head) {
        if(head == nullptr) return nullptr;
        return sortList(head,nullptr);
    }
    ListNode* sortList(ListNode* head,ListNode* tail){
        if(head->next == tail){
            head->next = nullptr;
            return head;
        }
        ListNode *slow = head,*fast = head;
        while(fast!=tail){
            slow = slow->next;
            fast = fast->next;
            if(fast!=tail){
                fast = fast->next;
            }
        }
        ListNode* mid =slow;
        return Merge(sortList(head,mid),sortList(mid,tail));
    }
    ListNode* Merge(ListNode* head1,ListNode* head2){
        ListNode* cur1 = head1;
        ListNode* cur2 = head2;
        
        ListNode* dummy = new ListNode();
        ListNode* cur = dummy;
        while(cur1&&cur2){
            if(cur1->val<=cur2->val){
                cur->next = cur1;
                cur1 = cur1->next;
            }
            else{
                cur->next = cur2;
                cur2 = cur2->next;
            }
            cur = cur->next;
        }
        if(cur1) cur->next = cur1;
        if(cur2) cur->next = cur2;
        cur = dummy->next;
        delete(dummy);
        return cur;
    }

};
```
### 🔥值得注意
如何使空间复杂度为O(1)呢？那只能自底向上的归并，即迭代，先归并大小为1，然后逐渐变大。代码还是有非常多细节的。还是分为三个部分

1.主函数，先算出链表长度，然后利用for循环控制步长，利用prev记录每次归并后的最后一个结点，以便连接后面的点，left为左边待排序的段，right为右边待排序的段，要注意的是，排序前他们最后一个节点都是与原链表断开的，并用cur记录下一段待排序的首节点，

后续再用prev接住排好序的两段！切割的话用的是split函数。

2.split函数， 参数为前段的首节点和切割的长度，注意边界控制，若能切完，则将对应最后节点与后续结点断开，否则返回空结点。  切之前记得保存后续第一个结点，因为要返回这个结点以便切割下一组。

3.merge函数，合并就不多说了。最后接没比较完的链表时可用三元运算符装b。

如果不想delete，那就在栈创建，格式为：ListNode dummy(0, head);方便的很，函数结束自动消除，用new的话在堆里创建，还要手动delete。
### 💻 C++ 自下而上归并代码实现
```cpp
class Solution {
public:
    ListNode* sortList(ListNode* head) {
        if(!head||!head->next) return head;
        int length = 0;
        ListNode* node = head;
        while (node) {
            length++;
            node = node->next;
        }
        ListNode dummy(0, head);
        for(int sz = 1;sz<length;sz<<=1){
            ListNode* prev = &dummy;
            ListNode* cur = dummy.next;
            while(cur){
                ListNode* left = cur;
                ListNode* right = split(left,sz);
                cur = split(right,sz);
                prev->next = Merge(left,right);
                while(prev->next){
                    prev = prev->next;
                }
            }
        }
        
     
        return dummy.next;
    }

private:   
    ListNode* split(ListNode* head,int len){
        ListNode* cur = head;
        while(--len&&cur){
            cur = cur->next;
        }
        if(!cur) return nullptr;
        ListNode* next = cur->next;
        cur->next = nullptr;
        return next;
    }
    ListNode* Merge(ListNode* head1,ListNode* head2){
        ListNode* cur1 = head1;
        ListNode* cur2 = head2;
        
        ListNode dummy =  ListNode(0);
        ListNode* cur = &dummy;
        while(cur1&&cur2){
            if(cur1->val<=cur2->val){
                cur->next = cur1;
                cur1 = cur1->next;
            }
            else{
                cur->next = cur2;
                cur2 = cur2->next;
            }
            cur = cur->next;
        }
        cur->next = cur1 ? cur1 : cur2;
      
        return dummy.next;
    }
};
```
