## 763.划分字母区间
- **标签：** 贪心
- **原题链接：** [点击跳转](https://leetcode.cn/problems/partition-labels/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
想到记录最后一次出现的位置和最早出现的位置，但总是想用区间去做，类似气球那道题，实际上构造一个区间的vector很繁琐，我们完全可以不构造。

只记录最后出现的位置，然后直接遍历字符串，就像跳跃游戏2一样，或者说bfs一样，当遍历的点刚好是边界时，说明这一串就是符合题意的，记录相关量，然后再更新，非常巧妙。

### 💻 C++ 模拟代码实现
```cpp
class Solution {
public:
    int jump(vector<int>& nums) {
        if(nums.size()==1) return 0;
        int index = 0;
        int count = 0;
        while(index<nums.size()-1){
            if(index+nums[index]>=nums.size()-1){
                count++;
                break;
            }
            int next_cover = -1;
            int new_index = -1;
            for(int i = 1;i<=nums[index]&&index+i<nums.size();i++){
                int step = index+i;
                int current_cover = step+nums[step];
                if(current_cover>next_cover){
                    next_cover = current_cover;
                    new_index = step;
                }
            }
            index = new_index;
            count++;
        }
        return count;
    }
};
```

### 💻 C++ 标准贪心代码实现
```cpp
class Solution {
public:
    int jump(vector<int>& nums) {
        int ans = 0;
        int cur_distance = 0;
        int next_distance = 0;
        for(int i = 0;i<nums.size()-1;i++){
            next_distance = max(i+nums[i],next_distance);
            if(i == cur_distance){
                ans++;
                cur_distance = next_distance;
            }
        }
        return ans;
    }
};
```
