## 763.划分字母区间
- **标签：** 贪心
- **原题链接：** [点击跳转](https://leetcode.cn/problems/partition-labels/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
想到记录最后一次出现的位置和最早出现的位置，但总是想用区间去做，类似气球那道题，实际上构造一个区间的vector很繁琐，我们完全可以不构造。

只记录最后出现的位置，然后直接遍历字符串，就像跳跃游戏2一样，或者说bfs一样，当遍历的点刚好是边界时，说明这一串就是符合题意的，记录相关量，然后再更新，非常巧妙。

### 💻 C++ 标准贪心代码实现
```cpp
class Solution {
public:
    vector<int> partitionLabels(string s) {
        vector<int> res;
        int hash[26] = {0};
        for(int i = 0;i<s.size();i++){
            hash[s[i]-'a'] = i;
        }
        int left = 0;
        int right ;
        for(int i = 0;i<s.size();i++){
            right = max(right,hash[s[i]-'a']);
            if(right == i){
                res.push_back(right-left+1);
                left = right+1;
            }
        }
        return res;
    }
};
```
