## 45.跳跃游戏Ⅱ
- **标签：** 贪心
- **原题链接：** [点击跳转](https://leetcode.cn/problems/jump-game-ii/description/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
只能说第二次做还是艰难，只能想到模拟，模拟的话边界条件也得很小心，模拟的思路是，从这个点出发，能达到的最有潜力的点是哪个，这就要求我们在遍历时，还得记录对应遍历到哪个点了，因为他随时可能是最有潜力的那个点，即更新最远边界。

当然，标准贪心也不难理解，而且代码量也少。当前这一步最多能覆盖一整个区间 ，在这个区间里，不急着立刻决定跳到哪一个点 ，只统计：这个区间里的所有点，谁能把下一层区间扩得最远，等走完整个当前区间，再统一把步数 +1；

很像bfs里求最短路径，自动就是最短步数了，我们不需要记录对应走到哪里了。但要注意的是，循环到 nums.size() - 1就可以了，因为到达最后一个位置就已经结束了，不需要再从最后一个位置往外跳。所以只要遍历到倒数第二个位置即可。

### 💻 C++ 模拟代码实现
```cpp
class Solution {
public:
    int jump(vector<int>& nums) {
        if(nums.size()==1) return 0;
        int index = 0;
        int count = 0;
        while(index<nums.size()-1){
            if(index+nums[index]>=nums.size()-1){
                count++;
                break;
            }
            int next_cover = -1;
            int new_index = -1;
            for(int i = 1;i<=nums[index]&&index+i<nums.size();i++){
                int step = index+i;
                int current_cover = step+nums[step];
                if(current_cover>next_cover){
                    next_cover = current_cover;
                    new_index = step;
                }
            }
            index = new_index;
            count++;
        }
        return count;
    }
};
```

### 💻 C++ 标准贪心代码实现
```cpp
class Solution {
public:
    int jump(vector<int>& nums) {
        int ans = 0;
        int cur_distance = 0;
        int next_distance = 0;
        for(int i = 0;i<nums.size()-1;i++){
            next_distance = max(i+nums[i],next_distance);
            if(i == cur_distance){
                ans++;
                cur_distance = next_distance;
            }
        }
        return ans;
    }
};
```
