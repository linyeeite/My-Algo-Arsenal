## 438. 找到字符串中所有字母异位词
- **标签：** 动态窗口 / 数组
- **原题链接：** [点击跳转](https://leetcode.cn/problems/find-all-anagrams-in-a-string/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
一开始想用sort，很明显会超时，然后想到了用26大小的哈希数组，但我还是每次都取出一个子串，然后重新算hash，导致时间复杂度很高，但也能ac，后面发现只需要加加减减就好了，利用p的大小和数组下标的随机访问还是很方便的。

后面还发现如果不是26个字母，是几万个unicode字符怎么办呢，如果每次都去完全比较会非常耗时，此时我们就维护一个diff变量就好了，就初始化的时候遍历两次，后续加加减减就好了。时间上再次优化！！
### 💻 C++ 代码实现
```cpp
class Solution {
public:
    vector<int> findAnagrams(string s, string p) {
        int n = p.size();
        vector<int> result;
        if(n > s.size()) return result;

        vector<int> p_count(26, 0);
        int diff = 0;

        // 1. 初始化第一个窗口
        for(int i = 0; i < n; i++){
            p_count[p[i] - 'a']++; // p 贡献正分
            p_count[s[i] - 'a']--; // s 贡献负分
        }

        // 2. 初始 diff 计算
        for(int a : p_count){
            if(a != 0) diff++;
        }

        if(diff == 0) result.push_back(0);

        // 3. 滑动窗口
        for(int i = n; i < s.size(); i++){
            // --- 处理新进入的 s[i]：它属于 s，所以应该执行 -- ---
            if(p_count[s[i] - 'a'] == 0) diff++;
            p_count[s[i] - 'a']--; 
            if(p_count[s[i] - 'a'] == 0) diff--;

            // --- 处理离开的 s[i-n]：它离开窗口，要把之前的 -- 补回来，所以执行 ++ ---
            if(p_count[s[i-n] - 'a'] == 0) diff++;
            p_count[s[i-n] - 'a']++; 
            if(p_count[s[i-n] - 'a'] == 0) diff--;

            if(diff == 0) result.push_back(i - n + 1);
        }
        return result;
    }
};
```
### 🔥值得注意
注意初始化时的逻辑得与后续移动的逻辑相同，即s中的字符贡献负值，就得一直贡献负值。还有如果要对比数组，直接对比就好了，无需还for循环一个一个对比
