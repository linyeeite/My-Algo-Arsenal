## 56.合并区间
- **标签：** 贪心/数组
- **原题链接：** [点击跳转](https://leetcode.cn/problems/merge-intervals/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
为了加快速度，肯定是要将数组排序，针对这种区间数组，我们肯定要自创cmp比较方式，然后记录和更新好对应区间left和right，逻辑见代码。我卡在了如何渐进把结果放进result里，想用while，实则用left和right，记录好相应区间大小，当区间合并不了时，放入即可。
### 💻 C++ 代码实现
```cpp
class Solution {
static bool cmp(const vector<int>& a,const vector<int>& b){
    return a[0]<b[0];
}
public:
    vector<vector<int>> merge(vector<vector<int>>& intervals) {
        if(intervals.size() == 1){
            return intervals;
        }
        vector<vector<int>> result;
        sort(intervals.begin(),intervals.end(),cmp);
        int left = intervals[0][0];
        int right = intervals[0][1];
        for(int i = 0;i<intervals.size()-1;i++){
            if(intervals[i+1][0]<=right){
                right = max(right,intervals[i+1][1]);
            }
            else {
                result.push_back({left,right});
                left = intervals[i+1][0];
                right = intervals[i+1][1];
            }
            if(i == intervals.size()-2){
                result.push_back({left,right});
            } 
        }
        return result;
    }
};
```
### 🔥值得注意
注意最后的边界控制，如果是倒数第二个，且已经跟最后一个比较后，直接放入result。
