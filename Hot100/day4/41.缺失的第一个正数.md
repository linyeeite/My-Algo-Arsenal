## 41.缺失的第一个正数
- **标签：** 贪心/数组
- **原题链接：** [点击跳转](https://leetcode.cn/problems/first-missing-positive/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
死去活来也无法将空间复杂度降为1，就算想到了用原数组作为哈希表也想不到swap和while的结合运用。难
### 💻 C++ 代码实现
```cpp
class Solution {
public:
    int firstMissingPositive(vector<int>& nums) {
        int n = nums.size();
        for(int i = 0;i<n;i++){
            while(nums[i]>0&&nums[i]<n+1&&nums[nums[i]-1]!=nums[i]){
                swap(nums[i],nums[nums[i]-1]);
            }
        }

        for(int i = 0;i<n;i++){
            if(nums[i]!=i+1) return i+1;
        }
        return n+1;
        
    }
};
```
### 🔥值得注意
注意条件控制即可，代码还是很短的。
