## 34.查找元素的第一和最后的位置
- **标签：** 二分查找/数组
- **原题链接：** [点击跳转](https://leetcode.cn/problems/find-first-and-last-position-of-element-in-sorted-array/description/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
做第二遍时，还是只能想到线性复杂度的做法，不知怎样更好地找到左边界和右边界，实际上，只需对标准二分法的逻辑稍加改动即可，我们不再是找到对应target的位置，而是找边界。既然找边界，对于找右边界，自然当mid对应的元素大于target时，right照常 = mid-1；

但当小于等于时，left = mid+1， 同时右边界rightboarder = left，即右边界是mid+1，你小于等于了，自然右边界是mid+1，又因为循环的判停条件是left>right，如果数组中存在小于target的值所以自然能找到右边界，若全大于target，**右边界不会更新**，则从理论上来说数组根本没有target，我们让他等于初始化值-2即可，后续用于判断即可。

找左边界同理，要义在于我们找的是开区间！所以注意主函数的条件判断。

### 💻 C++ 代码实现
```cpp
class Solution {
public:
    int get_right(vector<int>& nums, int target){
        int left = 0;
        int right = nums.size()-1;
        int rightboarder = -2;
        while(left<=right){
            int mid = left+(right-left)/2;
            if(nums[mid]>target) right = mid-1;
            else{
                left = mid+1;
                rightboarder = left;
            }
        }
        return rightboarder;
    }

    int get_left(vector<int>& nums, int target){
        int left = 0;
        int right = nums.size()-1;
        int leftboarder = -2;
        while(left<=right){
            int mid = left+(right-left)/2;
            if(nums[mid]<target) left = mid+1;
            else{
                right = mid-1;
                leftboarder = right;
            }
        }
        return leftboarder;
    }
    vector<int> searchRange(vector<int>& nums, int target) {
        int right = get_right(nums,target);
        int left = get_left(nums,target);
        if(left == -2||right == -2) return {-1,-1};
        if(right-left>1) return {++left,--right};
        else return{-1,-1};
    }
};
```
