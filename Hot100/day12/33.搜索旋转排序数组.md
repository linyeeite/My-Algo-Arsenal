## 33.搜索旋转排序数组
- **标签：** 二分查找/数组
- **原题链接：** [点击跳转](https://leetcode.cn/problems/search-in-rotated-sorted-array/description/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
二分查找的精髓在于“排除法”。在每一轮迭代中，我们先找到 mid，然后判断哪一半是“老实”的（即有序的）：

判定左半段是否有效：如果 nums[left] <= nums[mid]，说明左半段 [left, mid] 是严格升序的。

- 如果 target 恰好在这个有序区间内（nums[left] <= target < nums[mid]），那我们就去左边找：right = mid - 1。

- 否则，target 一定在不规则的右边：left = mid + 1。

判定右半段是否有效：如果上述条件不成立，说明右半段 [mid, right] 一定是严格升序的。

- 如果 target 在这个有序区间内（nums[mid] < target <= nums[right]），那我们就去右边找：left = mid + 1。

- 否则，target 一定在不规则的左边：right = mid - 1。

注意处理 left 和 mid 重合的情况（比如数组只有两个元素）。如果没有这个等号，逻辑会跳入 else 导致错误。
### 💻 C++ 代码实现
```cpp
class Solution {
public:
    int search(vector<int>& nums, int target) {
        int left = 0;
        int right = nums.size()-1;
        while(left<=right){
            int mid = left+(right-left)/2;
            if(nums[mid] == target) return mid;
            if(nums[left]<=nums[mid]){
                if(nums[left]<=target&&target<nums[mid]){
                    right = mid-1;
                }
                else left = mid+1;
            }
            else{
                if(nums[mid]<target&&target<=nums[right]){
                    left = mid+1;
                }
                else right = mid-1;
            }
        }
        return -1;
    }
};
```
