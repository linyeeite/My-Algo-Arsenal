## 73.矩阵置零
- **标签：** 矩阵 / 数组
- **原题链接：** [点击跳转](https://leetcode.cn/problems/set-matrix-zeroes/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
想不到原地算法，只能想到用标记位，实则可以利用第一行第一列作为标记位，为了防止丢失其他列其他行信息以及原本第一行第一列是否有零的信息，我们先用两个bool变量记录第一行第一列是否出现0，然后再遍历n-1*m-1矩阵，记录有0的行和列，即解。当然还有更优化的，只利用一个bool变量。有一个注意的点是，再标记完后，只需对各行各列遍历即可，时间很稳定，而不是记录对应行列坐标，然后再将一整行一整列变换为0，这样最坏情况会慢很多。
### 💻 C++ 代码实现
```cpp
class Solution {
public:
    void setZeroes(vector<vector<int>>& matrix) {
        int n = matrix.size();
        int m = matrix[0].size();
        bool row0 = false;
        bool col0 = false;
        for(int i = 0;i<n;i++){
            if(matrix[i][0]==0){
                col0 = true;
                break;
            }
        }
        for(int i = 0;i<m;i++){
            if(matrix[0][i]==0){
                row0 = true;
                break;
            }
        }
        for (int i = 1; i < n; i++) {
            for (int j = 1; j < m; j++) {
                if (matrix[i][j] == 0) {
                    matrix[i][0] = 0;
                    matrix[0][j] = 0;
                }
            }
        }
        for (int i = 1; i < n; i++) {
            for (int j = 1; j < m; j++) {
                if (matrix[i][0] == 0 || matrix[0][j] == 0) {
                    matrix[i][j] = 0;
                }
            }
        }
        if (row0) for (int j = 0; j < m; j++) matrix[0][j] = 0;
        if (col0) for (int i = 0; i < n; i++) matrix[i][0] = 0;
        
    }
};
```
### 🔥值得注意
一个标记量的思路是，还是利用第一行第一列记录0，但是利用第一列的第一个元素作为第一行是否有零的标记位，因为有一个标记量早已记录第一列是否有零，所以不用担心信息丢失。不同的地方在于，记录信息遍历的时候，之前跳过了第一行，这次要遍历第一行！还有个
很关键的是，在改变矩阵时，要从最后一行且第二列开始，因为第一行记录的是下面的有无零的信息，先改变就寄了。

### 💻 C++ 优化代码实现，优雅
``` cpp
class Solution {
public:
    void setZeroes(vector<vector<int>>& matrix) {
        int n = matrix.size();
        int m = matrix[0].size();
        
        bool col0 = false;
        for(int i = 0;i<n;i++){
            if(matrix[i][0]==0){
                col0 = true; 
            }
            for (int j = 1; j < m; j++) {
                if (matrix[i][j] == 0) {
                    matrix[i][0] = 0;
                    matrix[0][j] = 0;
                }
            }
         }
        for (int i = n-1; i>=0; i--) {
            for (int j = 1; j < m; j++) {
                if (matrix[i][0] == 0 || matrix[0][j] == 0) {
                    matrix[i][j] = 0;
                }
            }
            if(col0) matrix[i][0] = 0;
        }
        
    }
};
```
