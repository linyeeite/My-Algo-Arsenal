## 54.螺旋矩阵
- **标签：** 矩阵 / 数组
- **原题链接：** [点击跳转](https://leetcode.cn/problems/spiral-matrix/solutions/275393/luo-xuan-ju-zhen-by-leetcode-solution/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
一直在想代码随想录的不变量做法，首先那个是螺旋矩阵Ⅱ，方阵且要将loop的次数弄出来，还要处理奇偶的情况，个人认为过于复杂，很容易淡忘，相反，不去在乎循环不变量，只在乎边界，判断好是否越界，去模拟，而且代码很规整，逻辑也很清晰。
```cpp
class Solution {
public:
    vector<int> spiralOrder(vector<vector<int>>& matrix) {
        vector<int> result;
        int bottom = matrix.size()-1;
        int right = matrix[0].size()-1;
        int left = 0, top = 0;
        while(1){
            for(int i = left;i<=right;i++) result.push_back(matrix[top][i]);
            if(++top>bottom) break;
            for(int i = top;i<=bottom;i++) result.push_back(matrix[i][right]);
            if(--right<left) break;
            for(int i = right;i>=left;i--) result.push_back(matrix[bottom][i]);
            if(--bottom<top) break;
            for(int i = bottom;i>=top;i--) result.push_back(matrix[i][left]);
            if(++left>right) break;
        }
        return result;
    }
};
```
### 🔥值得注意
对于螺旋矩阵Ⅱ，只需将result.push_back改为对应的count++即可。
