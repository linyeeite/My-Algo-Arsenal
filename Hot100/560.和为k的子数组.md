## 560.和为k的子数组
- **标签：** 子串 / 数组
- **原题链接：** [点击跳转](https://leetcode.cn/problems/subarray-sum-equals-k/description/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
思来想去，还是只能暴力，但还是超时了。看了题解，原来是用前缀和，从前向后的遍历顺序和前缀和保证了是题目所要求的子数组，非常巧妙，再利用map记录对应前缀和，相加就好了，因为只用记录次数，无需记录下标，++即可。
### 💻 C++ 代码实现
```cpp
class Solution {
public:
    int subarraySum(vector<int>& nums, int k) {
        int result = 0;
        int presum  = 0;
        unordered_map<int,int> umap;
        umap[0] = 1;
        for(int x : nums){
            presum+=x;
            if(umap.find(presum-k)!=umap.end()){
                result+=umap[presum-k];
            }
            umap[presum]++;
        }
        return result;

    }
};
```
