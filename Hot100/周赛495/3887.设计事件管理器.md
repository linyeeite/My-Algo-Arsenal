## 3887.设计事件管理器
- **标签：** set的运用/懒更新堆
- **原题链接：** [点击跳转](https://leetcode.cn/problems/count-good-subarrays/description/)

### 💡 思路历程
一开始的思路就是用vector + 排序，时间复杂度爆了，后来发现平时用多了unodered_set，都忘记set是自动排序的了，底层是红黑树，每次插入，删除，更新的时间复杂度都是log n。因为set的默认排序是递增的，所以我们先把priority排在pair前面，然后id放后面。

又因为相同priority的id越小的优先级越高，所以我们传入id的相反数，然后每次用set.rbegin()取set最后一个元素，因为end（）取得是后一个位置，同理，可以理解begin（），rend（）。同时，set只可insert()or emplace(),不可push();



### 💻 C++ set代码实现
```cpp
#include <vector>
#include <unordered_map>
#include <set>
#include <algorithm>

using namespace std;

class EventManager {
private:
    unordered_map<int, int> idToPriority;
    // {priority, id} 的有序集合，默认升序
    // 我们利用默认排序：优先级升序；优先级相同时，ID 升序
    set<pair<int, int>> st;

public:
    EventManager(vector<vector<int>>& events) {
        for (auto& e : events) {
            int id = e[0], priority = e[1];
            idToPriority[id] = priority;
            st.insert({priority, -id}); // ID取负是为了让“小ID”在“大优先级”里排在后面
        }
    }
    
    void updatePriority(int eventId, int newPriority) {
        if (idToPriority.count(eventId)) {
            int oldPriority = idToPriority[eventId];
            // 1. 从集合中删除旧状态 O(log N)
            st.erase({oldPriority, -eventId});
            idToPriority[eventId] = newPriority;
            st.insert({newPriority, -eventId});
        }
    }
    
    int pollHighest() {
        if (st.empty()) return -1;

        // 4. 取出最大的元素（末尾） O(1)
        auto it = st.rbegin(); 
        int id = -(it->second); // 取反回来得到原始 ID
        
        st.erase(prev(st.end()));
        idToPriority.erase(id);
        
        return id;
    }
};
```
### 🔥值得注意
上面的方法还是有些暴力，我们还有懒更新堆法，核心思路是我不去删除旧的数据，只是在读取时筛选过失的信息。
### 💻 C++ 懒更新堆代码实现
```cpp
class EventManager {  
private:
    unordered_map<int, int> idToPriority;
    priority_queue<pair<int,int>> pq;
public:
    EventManager(vector<vector<int>>& events) {
        for (auto& e : events) {
            int id = e[0], priority = e[1];
            idToPriority[id] = priority;
            pq.push({priority,-id});
        }
    }
    
    void updatePriority(int eventId, int newPriority) {
        if(idToPriority.count(eventId)){
            idToPriority[eventId] = newPriority;
            pq.push({newPriority,-eventId});
        }
    }
    
    int pollHighest() {
        while(!pq.empty()){
            auto [priority,id] = pq.top();
            pq.pop();
            if(idToPriority[-id] != priority){
                continue;
            }
            idToPriority.erase(-id);
            return -id;
        }
        return -1;
    }
};

```
