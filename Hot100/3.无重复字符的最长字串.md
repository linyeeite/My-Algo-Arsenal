## 3. 无重复字符的最长字串
- **标签：** 动态窗口 / 数组
- **原题链接：** [点击跳转](https://leetcode.cn/problems/longest-substring-without-repeating-characters/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
一开始想用unordered_set，但不记得erase（）了，且左指针的移动逻辑不是很清楚，还是看了题解，主要是对动态窗口不熟悉。用map是比set快的。原因在于重复太多erase，insert；

### 💻 C++ 代码实现
```cpp
class Solution {
public:
    int lengthOfLongestSubstring(string s) {
        int result  = 0;
        unordered_map<char,int> umap;
        for(int i = 0,j=0;i<s.size();i++){
            umap[s[i]]++;
            while(umap[s[i]]>1){
                umap[s[j]]--;
                j++;
            }
            result = max(result,i-j+1);
        }
        return result;
    }
};
```
