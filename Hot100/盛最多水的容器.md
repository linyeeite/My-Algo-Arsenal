## 11. 盛最多水的容器
- **标签：** 双指针 / 数组
- **原题链接：** [点击跳转](https://leetcode.cn/problems/container-with-most-water/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
想了许久，没想到先移动高度较小所对应的下标，而是采取了剪枝的操作，跳过矮柱子，其实是一种优化，但先移动高度较小的一边才是核心。

### 💻 C++ 代码实现
```cpp
class Solution {
public:
    int maxArea(vector<int>& height) {
        int result = 0;
        int left = 0,right = height.size()-1;
        while(left<right){
            int count = (right-left)*min(height[left],height[right]);
            if(height[left]<height[right]){
                left++;
            }
            else right--;
            result = max(count,result);
        }
        
        return result;
    }
};
```
### 值得注意
阐述下一开始的剪枝想法，在移动的时候，如果比一开始的柱子还矮，就得继续移动，即
```cpp
int j=1
while((right-j)>left&&height[right-j]<=height[right]) j++;
right = right-j;
```
去寻找对应位置，不过得要个边界判断，否则会越界。
### 💻 C++ 剪枝代码实现（常数级别的优化）
```cpp
class Solution {
public:
    int maxArea(vector<int>& height) {
        int result = 0;
        int left = 0,right = height.size()-1;
        while(left<right){
            int count = (right-left)*min(height[left],height[right]);
            if(height[left]<height[right]){
                int j=1;
                while((left+j)<right&&height[left+j]<=height[left]) j++;
                left = left+j;
            }
            else {
                int j=1;
                while((right-j)>left&&height[right-j]<=height[right]) j++;
                right = right-j;   
            }
            result = max(count,result);
        }
        
        return result;
    }
};
```
