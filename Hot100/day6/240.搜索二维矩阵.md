## 234.回文链表

- **原题链接：** [点击跳转](https://leetcode.cn/problems/palindrome-linked-list/submissions/711400962/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
重操旧题，第一时间想到的还是转换成数组然后双指针判断是否为回文，却想不起来找中点然后逆置再一一对比  😔
### 💻 C++ 代码实现
```cpp
class Solution {
public:
    bool isPalindrome(ListNode* head) {
        if(head == nullptr) return true;
        if(head->next == nullptr) return true;
        ListNode* slow = head;
        ListNode* fast = head;
        while(fast->next != nullptr&&fast->next->next != nullptr){
            slow = slow->next;
            fast = fast->next->next;
        }
        ListNode* pre = slow->next;
        slow->next = nullptr;
        pre = reverselist(pre);
        ListNode* cur = head;
        while(pre!=nullptr){
            if(pre->val!=cur->val) return false;
            pre = pre->next;
            cur = cur->next;
        }
        return true;
    }

    ListNode* reverselist(ListNode* head){
        ListNode* dummy = new ListNode(0);
        dummy->next = nullptr;
        ListNode* cur = head;
        while(cur!=nullptr){
            cur = cur->next;
            head->next = dummy->next;
            dummy->next = head;
            head = cur;
        }
        head = dummy->next;
        delete(dummy);
        return head;
    }
};
```
### 🔥值得注意
reverse可以不用虚头结点，拿三个指针即可，然后注意中点在前半段链表里。
