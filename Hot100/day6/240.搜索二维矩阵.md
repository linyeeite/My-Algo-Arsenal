## 240.搜索二维矩阵

- **原题链接：** [点击跳转](https://leetcode.cn/problems/search-a-2d-matrix-ii/solutions/1062538/sou-suo-er-wei-ju-zhen-ii-by-leetcode-so-9hcx/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
一开始想的是从左往右遍历，所以只能想到对列的剪枝（时机为遍历的元素大于target），也想过用二分法，但不知道调用lower_bound(row.begin(), row.end(), target)函数，这个函数查找的是第一个不小于target对应元素的指针。但最好的方法是同时利用行和列的单调，即从
右上角开始，大于target就对列剪枝，小于target就对行剪枝，因为这是最右上元素的特殊性，看图就好理解了。
### 💻 C++ 代码实现
```cpp
class Solution {
public:
    bool searchMatrix(vector<vector<int>>& matrix, int target) {
        int m = 0;
        int n = matrix[0].size()-1;
        while(m<matrix.size()&&n>=0){
            if(matrix[m][n] == target) return true;
            else if(matrix[m][n] > target) n--;
            else m++;
        }
        return false; 

    }
};
```
