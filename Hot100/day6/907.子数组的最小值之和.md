## 907.子数组的最小值之和
- **标签：** 单调栈 / 数组
- **原题链接：** [点击跳转](https://leetcode.cn/problems/sum-of-subarray-minimums/description/)

### 💡 思路历程
只能想到暴力，暴力的话，n方的复杂度。其实本质上可以找左右两边第一个比自己小的数,那自然想到单调栈！然后利用乘法原理算出对应子数组个数然后乘以本身大小得到贡献，当然，里面涉及了去重操作，一边找小于，一边找小于等于，还是很巧妙的。

从细节出发，又有遍历三次，遍历两次，遍历一次，很是风骚 🤡  

一次遍历的话，得在栈底加个哨兵，且在数组后也得加个-1，为了栈中要出贡献的元素全部弹出来。对单调栈的理解要很深刻。
### 💻 C++ 一次遍历代码实现
```cpp
class Solution {
    const int MOD = 1e9 + 7;
public:
    int sumSubarrayMins(vector<int>& arr) {
        arr.push_back(-1);
        long ans = 0;
        stack<int> st;
        st.push(-1);
        for(int i = 0;i<arr.size();i++){
            while(st.size()>1&&arr[st.top()]>=arr[i]){
                int k = st.top();
                st.pop();
                ans+=(long)arr[k]*(k-st.top())*(i-k);
            }
            
            st.push(i);
        }
        return ans%MOD;
    }
};
```
