## 207.课程表
- **标签：** 图论
- **原题链接：** [点击跳转](https://leetcode.cn/problems/course-schedule/description/?envType=study-plan-v2&envId=top-100-liked)

### 💡 思路历程
一开始理解成了判环，用并查集，事实上并查集只好做无向图的判环，无法判断有向图是否成环，所以本题实际上考察的是**拓扑排序**。分为bfs和dfs两种，本次再分别阐释一波思路。

🔥**bfs**：

1.从入度为0的点开始，所以我们需要一个indgree数组循环一次图统计下入度，同时将原始的只有边的图转为为邻接表adj。

2.创建队列，将入度为0点的点放入队列，利用count记录从队列里弹出的点数量，用于后面判断是否能进行完整的拓扑排序。

3.去除front点，遍历adj将对应indgree减一，只有减一的点有可能成为入度为0的点，判断是否能入队列。常规bfs思路。

4.看count值是否等于点数，等于即可以完成拓扑排序，否则则不行。

### 💻 C++ bfs代码实现
```cpp
 class Solution {
public:
    bool canFinish(int numCourses, vector<vector<int>>& prerequisites) {
       vector<int> indegree(numCourses,0);
       vector<vector<int>> adj(numCourses);
       for (auto& p : prerequisites) {
            int cur = p[0];
            int pre = p[1];
            adj[pre].push_back(cur); 
            indegree[cur]++;         
        }
       queue<int> que;
       int count = 0;
       for(int i = 0;i<numCourses;i++){
            if(indegree[i]==0){
                que.push(i);
                count++;
            }
       }
       while(!que.empty()){
            int cur = que.front();
            que.pop();
            for(int j : adj[cur]){
                indegree[j]--;
                if(indegree[j]==0) {
                    que.push(j);
                    count++;
                }
            }
       }
        return count==numCourses;

    }
};
```
🔥**dfs**：

1.dfs则是我们是“一路向深处”走，直到走到尽头（没有后继节点的节点），然后倒着把路径记录下来（如果需要的话）。

2.关键在于有三种状态，我们需要一个vector<int> visited 数组来记录对应状态，状态0对应未被访问，状态1对应正在递归的过程中，状态2对应已被访问完。

3.如若递归的过程中碰到了状态为1的点，说明成环，直接返回false即可。

4.还是有细节需要注意，比如主函数里，是对为访问过的点进行dfs，别忘记条件判断。还有外层循环的剪枝判断，如果成环，就没必要继续下去了，这一点再dfs函数里也有，比如：if(hasCycle) return;

### 💻 C++ dfs代码实现
```cpp
class Solution {
private:
    vector<int> visited;
    vector<vector<int>> adj;
    bool hasCycle = false;
public:
    void dfs(int course){
        visited[course] = 1;
        for(int i : adj[course]){
            if(visited[i] == 0){
                dfs(i);
                if(hasCycle) return;
            }
            else if(visited[i] == 1){
                hasCycle = true;
                return;
            }
        }
        visited[course] = 2;
    }

    bool canFinish(int numCourses, vector<vector<int>>& prerequisites) {
        visited.resize(numCourses,0);
        adj.resize(numCourses);
        for(auto& a : prerequisites){
            adj[a[1]].emplace_back(a[0]);
        }
        //注意下面的!hasCycle，剪枝操作
        for(int i = 0;i<numCourses&&!hasCycle;i++){
            //一定不能漏这个条件判断！
            if(visited[i] == 0){
                dfs(i);
            }
        }
        return !hasCycle;
    }
};
```
