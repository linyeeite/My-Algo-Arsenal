## 3878.统计好子数组
- **标签：** logtrick / 数组
- **原题链接：** [点击跳转](https://leetcode.cn/problems/count-good-subarrays/description/)

### 💡 思路历程
这道题太有难度，可谓是毫无思路可言，首先得对或运算这种单调不减的运算很敏感，并用集合去理解，我们要利用vector<pair<int,int>>去记录对应or值和右端点下标，这是一定得记录的，否则我们无法统计对应子数组是否满足“好”的要求。

然后就是对该集合的去重，因为单调不减的性质，很可能做完位运算后，or值是不变的，我们没有必要记录每个子数组的or值，否则后续或运算就是O（n）的时间复杂度，采用去重逻辑，如若or值相同，记录最左边的下标即可，可见t26。

要清晰这个or值的动态变化过程，我们一开始从最左边的元素开始，依次与上轮得到的or值进行或运算然后去重，此时得到的就是本轮不同or值的集合，相当于我们把信息压缩了，我们能够根据这个集合中的信息判断出以本轮元素为right边界时对应不同or值对应的left边界。

若不清晰，用一个{4，2，3，1}的例子自己手推一下就好理解多了。

然后就是对子数组进行判断，其实就是对应的or值是否出现在对应区间中，我们在从左向右遍历得过程中，利用一个umap来记录对应元素最后一次出现位置对应下标，用这个下标indicator与对应左边界做比较，如果在右边，自然是好数组，在左边，取不到，故不是。
我在这里一开始一直有个疑问，就是如果indicator大于本轮元素下标了怎么办，那就不能算是一个好数组了呀！其实不会出现这种情况，因为遍历到本元素时，我们就开始做判断了，无论如何or值对应的下标都不会到后面去！

然后就是结果的累加，还是有点绕的。ans+=min（right，对应indicator）-left+1。 模拟一下就好了。
### 💻 C++ logtrick代码实现
```cpp
class Solution {
public:
    long long countGoodSubarrays(vector<int>& nums) {
        vector<pair<int,int>> or_left;
        unordered_map<int,int> last;
        long long ans = 0;
        for(int i = 0;i<nums.size();i++){
            int x = nums[i];
            last[x] = i;

            for(auto& [or_val, _] : or_left){
                or_val |= x;
            }
            or_left.emplace_back(x,i);

            int m = 1;
            for(int j = 1;j<or_left.size();j++){
                if(or_left[j].first!=or_left[j-1].first){
                    or_left[m++] = or_left[j];
                }
            }
            or_left.resize(m);

            for(int k = 0;k<m;k++){
                auto [or_val,left] = or_left[k];
                int right = k+1<m?or_left[k+1].second-1:i;
                auto it = last.find(or_val);
                if(it!=last.end()&&left<=it->second){
                    ans += min(right,it->second)-left+1;
                }
            }
        }
        return ans;
    }
};
```
### 值得注意
代码还有诸多细节，例如怎么从压缩的信息list中读取对应right值（前面的元素和最后一个元素是不一样的！），还有or_left每轮都会新增单只有本轮元素的一个集合，后续再去重。还有_占位符的运用。

还有单调栈+贡献法，用left数组记录左边第一个不在本集合里的。然后再反向遍历。
### 💻 C++ 单调栈代码实现（灵神）
```cpp
class Solution {
public:
    long long countGoodSubarrays(vector<int>& nums) {
        int n = nums.size();
        vector<int> left(n);
        stack<int> st;
        st.push(-1); // 哨兵
        for (int i = 0; i < n; i++) {
            int x = nums[i];
            while (st.size() > 1 && (nums[st.top()] | x) == x) {
                st.pop();
            }
            left[i] = st.top(); // nums[left[i]] 不是 x 的子集
            st.push(i);
        }

        st = stack<int>();
        st.push(n);
        long long ans = 0;
        for (int i = n - 1; i >= 0; i--) {
            int x = nums[i];
            // 比如 nums = [...,1,...,1,...]，我们规定，包含左边的 1 的子数组，不能包含右边的 1，从而避免重复统计子数组
            // 注：包含右边的 1 的子数组，可以包含左边的 1
            while (st.size() > 1 && nums[st.top()] != x && (nums[st.top()] | x) == x) {
                st.pop();
            }
            int right = st.top(); // nums[right] 不是 x 的子集
            st.push(i);

            // 子数组左端点可以从 left[i]+1 到 i，一共 i-left[i] 个
            // 子数组右端点可以从 i 到 right-1，一共 right-i 个
            ans += 1LL * (i - left[i]) * (right - i);
        }

        return ans;
    }
};

```
